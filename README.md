# Cloud Monitoring System 🚀

## 📌 Project Overview
This is a cloud-based monitoring system that simulates resource usage and displays system status (SAFE / ALERT).

## ⚙️ Tech Stack
- Python (Flask)
- Docker
- GitHub Actions (CI/CD)
- Render (Deployment)

## 🔄 CI/CD Pipeline
- Code push to GitHub triggers GitHub Actions
- Docker image is built automatically
- Application is deployed on Render

## 🐳 Docker
The application is containerized using Docker for consistent deployment.

## 📷 Output
Displays:
- Current Usage
- Status (SAFE / ALERT)
